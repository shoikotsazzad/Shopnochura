# Project Context

This is a 3D animated restaurant website project built with React.jsx and Tailwind CSS — frontend only. The goal is a visually stunning, modern restaurant website with smooth 3D animations and beautiful UI.

# Tech Stack

- **React.jsx** — Component structure
- **Tailwind CSS** — Styling
- **Three.js / React Three Fiber** — 3D elements and animations
- **Framer Motion** — Page transitions and UI animations
- **GSAP** — Scroll-based animations and timelines
